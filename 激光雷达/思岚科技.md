* 1. 创建ROS Catkin Workspace：

```html
# 新建并编译ROS工作空间:
mkdir -p ~/ws/src
cd ~/ws
catkin_make
```

* 2. 在catkin工作空间的src目录下，下载官方package：
```html
cd ~/ws/src
git clone https://github.com/Slamtec/rplidar_ros.git
```

* 3. catkin_make
```html
cd ~/ws
catkin_make
```

* 4. launch
```html
roslaunch rplidar_ros view_rplidar_a3.launch
# 备注：若不能连上/dev/ttyUSB0，参考之前解决方法。
```
